About the Project

This is a digital platform that uses AI to provide personalized therapy sessions for individuals seeking alternative forms of therapy. Users complete a questionnaire, and the AI system generates a customized therapy plan tailored to their unique needs and goals. The product is designed to provide an innovative and effective alternative to traditional therapy sessions, using the power of AI to deliver personalized, accessible, and effective therapy.

Features

  • Feature 1: Barebones frontend that handles sessions, sends user form data and conversation input to backend, and recieves AI output from backend.
  • Feature 2: Barebones backend that handles form data for a personalized experience, manages the AI input/output, encrypts session data, and deletes session data.
  • Feature 3: ProductHunt landing page for community forum.
  • Feature 4: RedisDB that stores encrypted session data temporarily.

Technologies Used

The project is built using the following technologies:

  • Technology 1: Python (FastAPI) is the language used for the backend
  • Technology 2: Redis is the technology used for the data store
  • Technology 3: JavaScript+HTML is the technology used for the frontend
